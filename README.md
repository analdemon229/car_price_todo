# Car Restoration Management System

Система для расчёта, документирования и управления задачами по восстановлению автомобиля после ДТП.

## Описание

Монорепозиторий, содержащий три приложения:

- **Backend** (NestJS) - REST API сервер
- **Desktop Client** (Angular) - Web-приложение для ПК
- **Android Client** (Android/Kotlin) - Мобильное приложение для Android
- **Ios Client** (Ios/Swift) - Мобильное приложение для Ios

Все приложения синхронизируются через единую облачную базу данных PostgreSQL, что позволяет иметь доступ к одним и тем же данным как на компьютере, так и на телефоне.

## Назначение

Система предназначена для:

- Планирования и отслеживания работ по восстановлению автомобиля
- Контроля расходов и составления смет
- Управления списком необходимых запчастей
- Документирования процесса восстановления
- Анализа прогресса и финансовых затрат

## Функционал

### Управление задачами

- Создание и редактирование задач восстановления
- Категории задач (кузовные работы, покраска, механика, электрика, салон)
- Приоритеты и статусы выполнения
- Зависимости между задачами
- Чек-листы для сложных задач

### Финансовый учёт

- Сметы по задачам
- Фактические расходы
- Сравнение плана и факта
- Категории расходов (запчасти, работа, материалы)
- Графики и аналитика (В разработке)

### Управление запчастями

- Список необходимых запчастей
- Статусы заказа (нужна, заказана, получена, установлена)
- Поставщики и цены
- Артикулы и каталожные номера

### Документирование

- Загрузка документов (акты, сметы, чеки)
- История изменений
- Комментарии и заметки
- Экспорт отчётов (PDF/Excel) (В разработке)

### Календарь и планирование

- Календарь работ (В разработке)
- Планирование этапов восстановления
- Напоминания о важных датах (В разработке)
- История выполненных работ

### Аналитика и отчёты

- Прогресс восстановления (%) (В разработке)
- Диаграммы расходов (В разработке)
- Временная шкала проекта (В разработке)
- Статистика по категориям работ (В разработке)
